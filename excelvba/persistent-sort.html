Persistent Sort Order Tool

Save and restore the Sort Order of a Table in Excel.

It is available on Github at https://github.com/ceybers/excel-persistent-sort.

What does it do?

Save the Custom Sort order of a Table in Excel so that you can restore it in future.

Why did you make this tool?

Sometimes Excel will reset the Sort Order of the columns in the Table you are working with, which is particularly annoying when you are sorting by several columns. This tool lets you save and restore the sort order, and those saved states persist across closing the file.

If you have multiple tables or multiple files with the same columns you want to sort by, you can apply the sort order to them, even if they don't have all the columns that the original table had.
