Related Table Tool

Tool for speeding up adding multiple VLOOKUPs between two tables.

It is available on Github at https://github.com/ceybers/excel-related-table-tool.

What does it do?

Adds a custom menu item to the right click menu that will fill in a VLOOKUP formula to lookup and return values from another table.

Why did you make this tool?

It seemed like a good idea that would speed up having to create multiple VLOOKUPs. Unfortunately the entire concept ended up being very finicky and unintuitive in practice, so I abandoned it in favour of Table Transfer Tool.
